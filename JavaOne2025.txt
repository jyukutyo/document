Unveiling the Hidden Layers of Java Class Files: Beyond Bytecode

In this session, we will explore the intricate details of Java class files beyond the commonly discussed bytecode. A key focus will be on the invokedynamic instruction and the attributes that enable its advanced capabilities, such as facilitating lambda expressions and optimizing string concatenation. We will also delve into the nestmates feature, which streamlines the interaction between outer and inner classes, enhancing both performance and flexibility in class file structure. Whether you are a seasoned Java developer or new to understanding the depths of the JVM, this session will provide valuable insights into the hidden layers of Java class files.

Java in Japan: A Journey of Community, Culture, and Global Integration

このセッションでは、日本のJavaユーザグループがどのようにして現在のようにグローバルなJavaコミュニティと深く連携するようになったか、その過程で乗り越えてきた数々のチャレンジ、そしてその経験がJUGに参加する開発者に与えた影響について紹介します。地理的な隔たりや言語の壁など、日本特有の課題が存在する中で、Javaは日本国内で長い間人気を保っており、日本Javaユーザグループ（JJUG）は現在1万2千人以上の会員を誇ります。この基盤の上で、日本の開発者たちは、海外のコミュニティとの連携を強化し、課題を解決してきました。例えば、JJUGが主催するカンファレンス「JJUG CCC」では、英語のセッションが増加しており、また英語でのみ対話が行われるアンカンファレンス「JOnsen」も毎年開催されています。日本の温泉地でリラックスしながら国際的な対話を促進するこのイベントは、特に貴重な機会となっています。日本には私を含め5人のJava Championがおり、コミュニティで非常に重要な役割を果たしています。彼らの活動は、日本のJavaコミュニティが国際的に影響力を持つ一因となっています。私自身、JUGを通じてJavaOneを知り、様々な国の技術カンファレンスに参加し、時には英語で講演を行うこともありました。これらの経験は、コミュニティ内での積極的な活動を目の当たりにしたことから生まれたものです。こうした経験から得られた洞察が、他の地域や国々におけるコミュニティ活動の促進や、Javaの普及にどのように貢献できるかについても考察します。私たちの経験が、Javaの未来を担う新世代の開発者たちにとっての一助となれば幸いです。

In this session, we will explore how Java User Groups (JUGs) in Japan have successfully overcome various challenges to deeply integrate with the global Java community, and how these efforts have positively impacted developers within the JUGs. Despite the unique challenges posed by geographical distance and language barriers, Java has maintained strong popularity in Japan. The Japan Java User Group (JJUG) now boasts over 12,000 members. Building on this foundation, Japanese developers have worked hard to strengthen their collaboration with international communities and address these challenges. For example, JJUG’s flagship conference, JJUG CCC, features an increasing number of English-language sessions, and the annual "JOnsen" unconference, held entirely in English, offers a unique opportunity for international dialogue in the relaxing atmosphere of Japan’s famous hot springs.

Japan is also home to five Java Champions, including myself, who play a crucial role in the community. Their activities have contributed significantly to the growing international influence of the Japanese Java community. Personally, my involvement with JUGs introduced me to JavaOne, leading to opportunities to attend and speak at various technical conferences around the world, sometimes even in English. These experiences stem from witnessing the proactive engagement within the community. In this session, I’ll also share insights on how these experiences can contribute to fostering community activities and the spread of Java in other regions and countries. I hope our experiences will serve as inspiration and guidance for the next generation of Java developers.
